Next Steps & Elevar Recommendations
The PSO theme has been live ~2 weeks and is stable. The following actions are recommended to unlock further performance gains.
⚠️
Important — Duplicate Tracking Risk: All 3rd-party pixel scripts that are also flowing through Elevar must be removed or disabled to prevent data duplication across GA4, Meta, TikTok, Snapchat, Pinterest, and Reddit. Until this is done, data accuracy and page speed are both compromised.
Channel Current Status Required Action Notes
GA4 Potentially dual-tracking via Google & YouTube app + Elevar Remove native app Disable GA4 in Shopify's Google & YouTube channel
Meta (Facebook) Pixel possibly still active via Facebook & Instagram app Disable pixel Disable Meta pixel in Shopify's Facebook & Instagram app
TikTok Pixel may be running natively alongside Elevar Disable pixel Disable TikTok Pixel in Shopify app
Snapchat Elevar handling client-side; GTM tags still active Pause GTM tags All Snapchat GTM tags must be paused to prevent duplication
Pinterest Native Sales Channel app active Decision required Pinterest app cannot separate Catalog vs Pixel. Options: migrate Catalog to GoDataFeed, or keep native app for pixel only (no Elevar for Pinterest)
Reddit Tracking via Elevar Audit for duplication Remove any hardcoded Reddit pixel scripts from theme
GTM Containers Two GTM containers detected loading (GTM-MLX8N5L + 1 other) Audit needed Elevar loads GTM-MLX8N5L. Confirm if the second container is intentional and contains no duplicate tags
Friendbuy Theme code + custom pixel in use Confirm with Friendbuy Elevar cannot confirm necessity; contact Friendbuy support to verify both code paths are required
GTM Speed Optimizations (Elevar)
These GTM strategies can be applied independently of pixel removal to further improve performance.
1
Remove old and unused tags Audit GTM for tags that are no longer serving active campaigns or tracking needs. Fewer tags = faster container load.
2
Defer marketing tags to Window Loaded (or time-based trigger) Move tags that fire on pageview to fire on Window Loaded instead. For non-critical tags, add a time-delay trigger (e.g. 2000ms). Note: later firing can reduce data capture rates — balance accuracy vs speed.
3
Remove unused visibility triggers Visibility triggers run a check every second to detect on-screen elements. Remove any that aren't critical — they can measurably drag down TBT.
4
Remove visibility triggers from Google Optimize tests Live or draft A/B tests using visibility triggers are especially costly. Audit and clean up any stale Optimize experiments.
Room to grow

The PSO theme has delivered solid LCP and payload reductions, particularly on PDP (−696ms, −1.03MB) and PLP (−602ms). Once 3rd-party pixel duplication is resolved, TBT improvements and further score gains are expected across PDP, PLP, and Homepage — especially the Homepage, where TBT regressed slightly after the PSO deploy.
